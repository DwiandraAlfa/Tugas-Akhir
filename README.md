Deskripsi Proyek
Aplikasi ini merupakan sistem pencatatan dan pemantauan film atau anime yang memungkinkan pengguna untuk menambahkan, menampilkan, mengubah, dan menghapus data tontonan. Setiap data yang dicatat mencakup judul, jenis tayangan (Anime, Film, atau Series), status tontonan (Plan, Ongoing, atau Finished), progress episode jika statusnya Ongoing, serta poster sebagai gambar pendukung. Aplikasi ini dibuat menggunakan PHP Native dan MySQL sebagai basis data lokal. Fitur pencarian juga disediakan untuk memudahkan pengguna menemukan judul tertentu. Sistem ini dirancang sederhana namun tetap modern dan mudah digunakan.
